🧶 Atelier Martins — Loja de Bolsas de Crochê
Site vitrine para venda de bolsas artesanais de crochê, com catálogo de produtos e sistema de compra rápida via PIX e WhatsApp.

✨ Funcionalidades

🛍️ Catálogo de produtos com foto, nome e preço
⚡ Botão de Compra Rápida — abre modal com dados do PIX instantaneamente
💬 Integração com WhatsApp — redireciona o cliente direto para o contato da loja
📱 Layout responsivo adaptado para celular
🎨 Identidade visual própria com fontes e favicon personalizados


🛠️ Tecnologias

HTML5 — estrutura da página
CSS3 — estilização e layout responsivo
JavaScript — interatividade (modal de pagamento e integração WhatsApp)
Boxicons — ícones de redes sociais
Google Fonts — tipografia (Trispace + Playfair Display)


🛒 Como funciona a compra

Cliente acessa o site e visualiza o catálogo de bolsas
Clica em "Compra Rápida" no produto desejado
Um modal abre com os dados do PIX da loja
Após o pagamento, o cliente clica em "Falar no WhatsApp" e envia o comprovante
